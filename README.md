# TabVault Pro 🚀
A Minimalist & Privacy-Focused Session Manager for Chrome.

TabVault Pro is designed for power users who need a lightweight, no-nonsense way to manage their browser workspace. It focuses on manual session organization and silent failsafe protection, keeping your browser fast and your data safe.

✨ Key Features
Manual Vault: Save your current windows and tabs into named, organized groups.

True Lock: Protect your most important session groups from accidental deletion.

Crash Protection: Features an invisible 'Shadow Save' engine that automatically backups your state, allowing for easy restoration after a browser crash.

Data Portability: Export any session instantly to Markdown (for Notion/Obsidian), CSV (for Excel), or Plain Text.

🛡️ Privacy & Security
100% Local Storage: All your data stays on your machine using chrome.storage.local. No cloud, no tracking, and no external servers.

Minimal Permissions: We only ask for the permissions strictly necessary to manage your tabs and save your sessions (tabs, storage, windows).

Manifest V3: Built using the latest Chrome extension standards for better performance and security.

🛠️ Installation (Developer Mode)
Download this repository as a ZIP file and extract it to a folder.

Open Chrome and navigate to chrome://extensions/.

Enable Developer mode in the top-right corner.

Click Load unpacked and select the folder where you extracted the files.
